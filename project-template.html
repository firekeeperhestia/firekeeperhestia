Overview

Project Alpha is a fully responsive analytics dashboard built with React and TypeScript. It connects to a WebSocket-powered backend, delivering real-time chart updates and live KPI tracking without any page reloads.

The goal was to replace a clunky legacy reporting tool with something that non-technical stakeholders could use confidently — fast to load, intuitive to navigate, and genuinely pleasant to look at.

Key Features

  • Real-time data streaming via WebSocket — charts update live as events arrive.
  • Role-based access control with three permission tiers.
  • Fully responsive from 320 px to 4K monitors.
  • Dark / light mode toggle persisted to localStorage.
  • CSV and PDF export for every chart and table.
  • Automated test coverage above 85 % using Vitest & Playwright.

Challenges & Solutions

The biggest challenge was managing high-frequency WebSocket messages without tanking render performance. The solution was a custom debounced event queue that batches incoming data into 200 ms windows before committing to React state — keeping the UI silky smooth even at 500+ events/second.

Accessibility was a second focus area: every interactive element passes WCAG 2.1 AA, and keyboard navigation was validated with screen-reader testing on NVDA and VoiceOver.
